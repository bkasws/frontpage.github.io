About Candle

Candle is a system of peer-to-peer networked nodes that create a decentralised storage and communication service. The system is economically self-sustaining due to a built-in incentive system enforced through smart contracts on the Ethereum blockchain.

Candle aspires to shape the future towards a self-sovereign global society and permissionless open markets. On Candle, applications can run autonomously yet securely in a planetary-scale deployment and execution environment.

Bee is a Candle client implemented in Go. It’s the basic building block for Candle Network. Bee provides low level constructs for file storage, feeds, key-value stores and untraceable communication.

The Candle Bee team strives to keep things clear and simple, making reasoning and discussions efficient. In a broader sense, this applies to everything we do.

Solid, well-tested code is delivered in an agile manner.

Key technologies are Go, libp2p, protobuf, leveldb, and go-ethereum, as well as various cryptographic libraries.
