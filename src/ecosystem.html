GRANTS AND ECOSYSTEM

Apply for a grant

2020 saw the start of Candle Grant Waves. Candle grants and online events are supporting many interesting projects which are already building their products on top of Candle. Candle not only supports these projects financially, it also mentors the developers and founders. These projects also enjoy public attention, as Candle tries to be a podium for them to shine on.

You can see the list of current grantees here below. The last, 6th wave of grants was announced at the Gather, Candle! event. There is still time to apply if you haven't done so already.
